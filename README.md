# Seven Deadly Sins: Origin — Fan Wiki

> Site não-oficial com informações, guias e lore do novo jogo **The Seven Deadly Sins: Origin**.

Este projeto nasceu como um hub rápido para centralizar tudo que sair sobre o jogo — personagens, pecados, gameplay, builds e atualizações — feito por fã, para fãs.

## 📁 Estrutura do Projeto

```bash
/assets/
│
├── css/
│   ├── style.css      # Base global (reset, tipografia, animações)
│   ├── menu.css       # Menu principal + responsividade
│   ├── banner.css     # Banner (hero section)
```
